# SHAP
Esta sección esta dedicada a la evaluación de modelos mediante SHAP. Si bien el análisis se ha definido de forma robusta para poder realizarse sobre cualquier modelo, nos enfocaremos principalmente en los modelos de regresión y clasificación de redes neuronales.

Shap permite dar explicatibilidad a una gran variedad de modelos que a simple vista parecen ser cajas negras, analizando el impacto que tiene cada variable tanto a nivel local como a nivel global.

## Instrucciones de uso
- Descargar la carpeta y montarla en su entorno.
- Ejecutar los bloques de código de ```SHAP.ipynb```

El archivo data.csv cuenta con las variables explicativas ya estandarizadas, luego del tratamiento de datos realizado en el notebook original
Los archivos con terminación .h5 son los modelos de redes necesarios para usar con la librería SHAP
